# Fraud-Detection-Dashboard
I analyzed 10,000 financial transactions to identify fraudulent behavior using risk indicators such as device score, IP score, transaction amount, location, payment mode, and merchant category.

# Key insights of findings:
• ATM and Online payments have the highest fraud percentages.

• Fraud peaks around the 11th hour of the day.

• High-risk devices combined with high transaction amounts lead to strong fraud signals.

• Travel & Electronics categories contribute most to fraud attempts.
